What is Ha11y?

Ha11y allows users to discover whether or not a website will be accessible to them before even opening the website. Ha11y Google Chrome Extension scores and displays the accessibility score of websites directly in a google search tailored directly around the users disability. When users first signup, they can select their disability and see personally-tailored results; otherwise, there is a default score for general accessibility.
